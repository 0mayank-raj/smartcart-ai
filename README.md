🛒 SmartCart AI Dashboard

📌 Overview
An AI-powered Machine Learning application that analyzes customer behavior and provides personalized business strategies based on spending patterns.
This app segments customers using ML and helps businesses make smarter marketing decisions.

🌐 Live Demo
🔗 https://smartcart-ai-5564yfhmnc2smdrhzcxmci.streamlit.app/

🧠 Project Overview
This project uses a trained ML clustering model to segment customers based on:

Age
Annual Income
Total Spending

Users can input customer details and instantly get:
Customer segment
Behavioral insights
Business strategy recommendations


⚙️ Tech Stack
Python
Pandas
NumPy
Scikit-learn
Streamlit
Matplotlib


🤖 Machine Learning Model
Model Used: KMeans Clustering
Preprocessing:
Feature scaling using StandardScaler


Outputs:
Customer Segment (Budget / Premium / Balanced)
Business Strategy Recommendation
Spending Behavior Insights


🚀 Features
Interactive dashboard UI
Real-time customer segmentation
Smart business recommendations
Spending behavior analysis
Income vs Spending visualization
Deployable on Streamlit Cloud
